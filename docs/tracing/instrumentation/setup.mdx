---
title: "Setup"
description: "Install the Lemma SDK and send traces directly to the Lemma API"
---

The Lemma SDK sends trace JSON directly to the Lemma API.

<Steps>
  <Step title="Install the SDK">
    <Tabs>
      <Tab title="TypeScript">
        ```bash
        npm install @uselemma/tracing
        ```
      </Tab>
      <Tab title="Python">
        ```bash
        pip install uselemma-tracing
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Set environment variables">
    Find your API key and project ID in [Lemma project settings](https://platform.uselemma.ai).

    ```bash
    export LEMMA_API_KEY="lma_your_key_here"
    export LEMMA_PROJECT_ID="proj_your_id_here"
    ```

    <Note>
      The SDK sends to `https://api.uselemma.ai` by default. Pass `baseUrl` to `new Lemma({ baseUrl })` only when you need a custom endpoint.
    </Note>
  </Step>

  <Step title="Create a client">
    <Tabs>
      <Tab title="TypeScript">
        ```typescript
        import { Lemma } from "@uselemma/tracing";

        export const lemma = new Lemma();
        ```
      </Tab>
      <Tab title="Python">
        ```python
        from uselemma_tracing import Lemma

        lemma = Lemma()
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Send your first trace">
    <Tabs>
      <Tab title="TypeScript">
        ```typescript
        await lemma.trace(
          {
            name: "hello-agent",
            input: "What is the capital of France?",
          },
          async () => {
            return "Paris.";
          },
        );
        ```
      </Tab>
      <Tab title="Python">
        ```python
        lemma.trace(
            "hello-agent",
            lambda trace: "Paris.",
            input="What is the capital of France?",
        )
        ```
      </Tab>
    </Tabs>

    Open the Lemma dashboard and select **Traces**. Your trace appears at the top of the list.
  </Step>
</Steps>
