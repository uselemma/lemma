---
title: "Setup"
description: "Install the Lemma SDK and send traces directly to the Lemma API"
---

The Lemma SDK sends trace JSON directly to the Lemma API.

## 1. Install

<Tabs>
  <Tab title="TypeScript">
    ```bash
    npm install @uselemma/tracing
    ```
  </Tab>
  <Tab title="Python">
    ```bash
    pip install uselemma-tracing
    ```
  </Tab>
</Tabs>

## 2. Set environment variables

Find your API key and project ID in [Lemma project settings](https://platform.uselemma.ai).

```bash
export LEMMA_API_KEY="lma_..."
export LEMMA_PROJECT_ID="proj_..."
```

<Note>
  The SDK sends to `https://api.uselemma.ai` by default. Pass `baseUrl` to `new Lemma({ baseUrl })` only when you need a custom endpoint.
</Note>

## 3. Create a client

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    import { Lemma } from "@uselemma/tracing";

    export const lemma = new Lemma();
    ```
  </Tab>
  <Tab title="Python">
    ```python
    from uselemma_tracing import Lemma

    lemma = Lemma()
    ```
  </Tab>
</Tabs>

## 4. Send your first trace

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    await lemma.trace(
      {
        name: "hello-agent",
        input: "What is the capital of France?",
      },
      async () => {
        return "Paris.";
      },
    );
    ```
  </Tab>
  <Tab title="Python">
    ```python
    lemma.trace(
        "hello-agent",
        lambda trace: "Paris.",
        input="What is the capital of France?",
    )
    ```
  </Tab>
</Tabs>

Open the [Lemma dashboard](https://platform.uselemma.ai) -> **Traces**. Your trace appears within seconds.
