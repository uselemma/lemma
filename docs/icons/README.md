# Docs icons

These SVGs are [Remix Icon](https://remixicon.com/) glyphs (same set as `@remixicon/react` on the platform).

Mintlify only supports Font Awesome, Lucide, and Tabler as built-in libraries, so icons are vendored here and referenced as `/icons/<name>.svg` from `docs.json` and MDX.

When adding or replacing an icon, copy the matching Remix line glyph so docs stay visually aligned with the platform.
