The Lemma REST API lets you ingest traces, read runs, and manage projects and monitors directly. The interactive reference, generated from the OpenAPI specification, documents every endpoint, request schema, and response.

Browse the endpoints in the **API Reference** tab, or fetch the raw specification from [`https://api.uselemma.ai/openapi.json`](https://api.uselemma.ai/openapi.json).
