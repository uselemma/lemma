title: "Agent skill"

---

Use the Lemma agent skill when you want a coding agent to install the SDK, detect your framework, and add tracing for you.

The skill lives in the public Lemma developer repo:

```text
https://github.com/uselemma/lemma/tree/main/skills/lemma-tracing
```

## Install

```bash
npx skills add uselemma/lemma --skill "skills/lemma-tracing"
```

For Cursor:

```bash
npx skills add uselemma/lemma --skill "skills/lemma-tracing" --target cursor
```

For Claude Code:

```bash
npx skills add uselemma/lemma --skill "skills/lemma-tracing" --target claude
```

## Prompt your agent

After installing the skill, paste this into your coding agent:

```text
Use the Lemma AI skill from github.com/uselemma/lemma at skills/lemma-tracing to add tracing to this application.
```

The skill covers manual SDK tracing, Vercel AI SDK, OpenAI Agents SDK, LangChain, LangGraph, Langfuse side-by-side installs, trace handles, debug mode, and common tracing issues.
